C++运算符重载通用格式

2022-01-22
C++

摘自《C++ Primer》。

假设要在类Test上重载运算符。

输入输出运算符

输出运算符 <<

非成员函数，第一个形参为ostream引用，第二个形参为常量Test引用，返回ostream引用。

输入运算符 >>

非成员函数，第一个形参为istream引用，第二个形参为Test引用，返回istream引用。

输入运算符重载的实现中应该含有对输入错误的处理，尽可能保证对象的安全。

算术运算符

非成员函数，形参均为常量Test引用，返回Test对象。

一般会定义一个对应的复合赋值运算符，通常使用复合赋值来定义算术运算符。

关系运算符

相等运算符 ==

非成员函数，形参均为常量Test引用，返回布尔类型。

同时应利用相等运算符来定义不等运算符 !=

关系运算符 <

非成员函数，形参均为常量Test引用，返回布尔类型。
如果需要关系运算符，先定义顺序关系，然后通过顺序关系和相等关系定义其他关系。
顺序关系的逻辑应当符合相等关系：若两个对象都不小于对方，那么它们应当相等。

赋值运算符

拷贝赋值 =

成员函数，形参为常量Test引用，返回Test引用（左侧运算对象的引用）。

复合赋值

成员函数，形参为常量Test引用，返回Test引用（左侧运算对象的引用）。

下标运算符 []

成员函数，形参为索引类型，返回所访问元素的引用。
通常应该定义两个版本，分别返回普通引用和返回常量引用。

递增递减运算符

通常用于迭代器类。

前置递增递减

成员函数，返回递增或递减后对象的引用。

后置递增递减

成员函数，返回原值（不是引用）。

成员访问运算符

通常用于迭代器类和智能指针类。

解引用 *

成员函数，返回所指向对象的引用。

箭头 ->

成员函数，返回指向对象的指针。

UE4

TIndexedContainerIterator

一个通用的迭代器，要求容器具有以下接口：

  • 返回容器中对象总数
  • 返回给定索引是否有效
  • 重载了下标运算符
  • 能够删除给定索引的元素

模板参数

  • ContainerType 容器的类型
  • ElementType 容器中元素的类型
  • SizeType 容器的SizeType

构造参数

  • 容器对象引用, 迭代器起始索引

运算符重载

  • ++ -- += + -= -
  • * ->
  • == !=

成员函数

  • GetIndex 获取当前指向
  • Reset 将指向重置为0
  • SetToEnd 将指向设为尾部元素下一位置
  • RemoveCurrent 删除当前指向的元素

TCheckedPointerIterator

一个类似于指针的迭代器，用于检测范围for循环中容器大小是否改变。
